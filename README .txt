Чтобы запустить программу можно нажать на ярлык "Govorun", либо в папке "Govorun\govorunForm\bin\Debug\netcoreapp3.1" запустить "govorunForm.exe", либо открыть файл проекта в VS.

Исходные файлы вопросов и ответов содержатся в папке "sourse files", изменяемые в ходе работы программы файлы - непосредственно "Govorun\govorunForm\bin\Debug\netcoreapp3.1";

Чтобы вернуть программе исходное состояние следует заменить файлы в папке "Govorun\govorunForm\bin\Debug\netcoreapp3.1" файлами из папки "sourse files";
